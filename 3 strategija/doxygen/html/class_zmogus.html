Detailed Description

Abstrakti bazinė klasė, sauganti bendrus žmogaus duomenis.

Ši klasė tarnauja kaip pagrindas kitoms klasėms (pvz., Studentas). Saugo vardą ir pavardę, bei apibrėžia gryną virtualią funkciją print().
